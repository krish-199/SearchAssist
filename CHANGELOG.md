# Changelog

All notable changes to Search Assist will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.5] - 2026-03-06

### Current Release
- Material 3 Design implementation
- Edge swipe gesture detection
- Customizable edge width settings
- Accessibility service integration
- Search bar auto-detection in foreground apps
- Quick access trigger search functionality

### Features
- Edge swipe gesture to trigger search
- Customizable edge detection width
- Material 3 UI with Jetpack Compose
- Accessibility service for app interaction
- Boot receiver for persistent service

### Technical
- Minimum SDK: Android 8.0 (API 26)
- Target SDK: Android 15 (API 36)
- ProGuard optimization enabled
- Resource shrinking enabled
